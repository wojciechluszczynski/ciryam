# Ciryam

Oficjalna strona polskiego zespołu rockowego Ciryam — koncerty z ticketingiem, sklep z merchem, muzyka, galeria prasowa i aktualności. Dwujęzyczna (PL/EN).

**Live:** [ciryam.eu](https://ciryam.eu)

---

## Funkcje

- Kalendarz koncertów z mapą eventów i linkami do ticketingu
- Sklep — koszulki, winyle, merch z koszykiem i procesem zamówienia
- Odtwarzacz muzyki — sticky player z kontrolkami
- Galeria prasowa i wideo
- Instagram feed
- Dwujęzyczność PL/EN (przełącznik języka z kontekstem)
- Cookie banner + RODO
- Custom cursor
- Animowane liczniki

---

## Stack

| Warstwa | Technologia |
|---|---|
| Framework | React 18 + TypeScript |
| Build | Vite |
| UI | shadcn/ui + Radix UI |
| Styling | Tailwind CSS |
| Backend | Supabase |
| Hosting | Netlify |

---

## Lokalne uruchomienie

```bash
npm install
npm run dev
```

Aplikacja dostępna pod [http://localhost:5173](http://localhost:5173).
